# Control Statements

Control statements allow a program to make decisions and execute different blocks of code based on conditions.

## Types of Control Statements

| Statement | Description |
|-----------|-------------|
| if | Executes a block when a condition is true |
| if-else | Chooses between two blocks |
| else-if ladder | Checks multiple conditions |
| nested if | An if statement inside another if |
| switch | Selects one block from many options |
| enhanced switch | Modern switch expression (Java 14+) |
